# cvm-images

Reproducible CVM (Confidential Virtual Machine) image builds for the Lucia privacy-preserving AI agent platform.

## Contents

- Dockerfiles for agent CVM and inference TEE images
- Build scripts for reproducible builds
- CI configuration for build verification
- Golden image hashes for integrity validation

## About Lucia

Lucia is a privacy-preserving AI agent platform that delivers autonomous agent capabilities with hardware-enforced privacy via Trusted Execution Environments (TEE). Learn more at [luciaclaw.com](https://luciaclaw.com).
